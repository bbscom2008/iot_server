# 项目文件使用说明

## 📁 项目中的文件类型

项目中主要有两种测试/使用文件：

### 1. `.html` 文件（网页文件）
### 2. `.http` 文件（API测试文件）

---

## 🌐 HTML 文件使用说明

### 文件位置
```
src/main/resources/static/index.html
```

### 什么是 HTML 文件？
- HTML 文件是**网页文件**
- 用浏览器打开和访问
- 提供图形化的用户界面
- 可以直接操作，无需编程知识

### 如何使用 `index.html`？

#### 方法一：通过浏览器访问（推荐）

1. **启动 Spring Boot 应用**
   ```bash
   mvn spring-boot:run
   ```
   或在 IDE 中运行 `DemoApplication.java`

2. **打开浏览器访问**
   ```
   http://localhost:8080/index.html
   ```
   或者直接访问：
   ```
   http://localhost:8080/
   ```

3. **开始使用**
   - 界面会自动加载所有用户
   - 可以添加、编辑、删除用户
   - 所有操作都在网页上完成

#### 方法二：直接在 IDE 中预览

1. 在 IDE（如 IntelliJ IDEA）中右键点击 `index.html`
2. 选择 "Open in Browser" 或 "Preview"
3. **注意**：这种方式查看界面，但无法连接到后端（除非应用已启动）

### HTML 文件的特点

✅ **优点：**
- 图形化界面，易于使用
- 无需了解 API 格式
- 适合非技术人员使用
- 实时显示结果

❌ **局限：**
- 需要启动完整的应用
- 只能在浏览器中使用

## 🔌 HTTP 文件使用说明

### 文件位置
```
test_api.http           # 基本功能测试
test_crud_api.http      # CRUD 完整测试
```

### 什么是 .http 文件？
- `.http` 文件是 **HTTP 请求测试文件**
- 用于直接测试 API 接口
- 类似 Postman，但更轻量
- 专为开发者设计

### 支持 .http 文件的工具

1. **IntelliJ IDEA** (自带支持)
2. **VS Code** (需要安装插件: REST Client)
3. **WebStorm**
4. **PyCharm**

### 如何使用 .http 文件？

#### 在 IntelliJ IDEA 中使用

1. **打开文件**
   - 双击 `test_api.http` 或 `test_crud_api.http`

2. **执行请求**
   - 每个请求前有一个 **绿色播放按钮** ▶️
   - 点击按钮执行该请求
   - 或按快捷键：`Ctrl + Enter` (Windows) / `Cmd + Enter` (Mac)

3. **查看结果**
   - 结果会在右侧窗口显示
   - 包含状态码、响应头、响应体

#### 在 VS Code 中使用

1. **安装插件**
   - 打开扩展商店
   - 搜索 "REST Client"
   - 安装插件

2. **打开 .http 文件**
   - 打开 `test_api.http`

3. **执行请求**
   - 每个请求上方会显示 "Send Request"
   - 点击执行
   - 结果在新标签页显示

### .http 文件格式说明

```http
### 这是注释，描述这个请求的作用
GET http://localhost:8080/api/user/all

###

### 带请求体的POST请求
POST http://localhost:8080/api/user/register
Content-Type: application/json

{
  "username": "zhangsan",
  "password": "123456"
}

###
```

#### 格式要点：
- `###` 用于分隔不同的请求
- 第一行是 HTTP 方法和 URL
- 请求头在第二行开始（如 `Content-Type`）
- 空一行后是请求体（JSON 数据）

### HTTP 文件的特点

✅ **优点：**
- 快速测试 API 接口
- 无需打开浏览器
- 可以保存和分享测试用例
- 适合开发和调试

❌ **局限：**
- 需要了解 HTTP 协议
- 没有图形界面
- 适合技术人员使用

## 📊 对比：HTML vs HTTP 文件

| 特性 | HTML 文件 | HTTP 文件 |
|------|-----------|-----------|
| **用途** | 提供图形界面 | 测试 API 接口 |
| **使用工具** | 浏览器 | IDE (IDEA, VS Code) |
| **访问方式** | http://localhost:8080/index.html | 在 IDE 中直接执行 |
| **适用人群** | 所有用户 | 开发者 |
| **是否需要启动应用** | ✅ 是 | ✅ 是 |
| **图形界面** | ✅ 有 | ❌ 无 |
| **操作方式** | 点击按钮 | 执行 HTTP 请求 |

## 🔧 常见问题

### Q1: 为什么 HTML 页面打不开？
**A:** 确保：
- Spring Boot 应用已启动
- 访问地址正确：`http://localhost:8080/index.html`
- 端口 8080 没有被占用

### Q2: 为什么 .http 文件没有执行按钮？
**A:** 
- **IntelliJ IDEA**: 确保使用 Ultimate 版本（社区版也支持）
- **VS Code**: 需要安装 "REST Client" 插件
- 确保文件扩展名是 `.http`

### Q3: HTML 页面显示"加载失败"？
**A:** 
- 检查后端是否启动
- 打开浏览器控制台（F12）查看错误信息
- 确认 API 地址正确

### Q4: 可以同时使用 HTML 和 HTTP 文件吗？
**A:** 可以！
- HTML 用于日常操作
- HTTP 文件用于测试和调试
